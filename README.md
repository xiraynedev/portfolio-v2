## Installation Instructions

1. Clone the repo:

```sh
git clone https://github.com/xiraynedev/portfolio-v2.git
```

2. Change directory:

```sh
cd portfolio-v2
```

3. Install packages:

```sh
npm i
```

4. Start the dev server by running:

```sh
npm run dev
```
